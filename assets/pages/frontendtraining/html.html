What is HTML?

HTML is a markup language for creating website structure like inserting texts,links,images,lists,forms etc. HTML is also known as skeleton of websites. To create html file simply save file named with ".html" extention

What is HTML Tags?

HTML tags are a element which is surronded by lessthan "<" and greaterthen ">" symbol. This is the example for html tag < tagname> content </tagname>

HTML tags are two types:

1. Paired Tags: Paired tag mean it have both the opening tags "< tagname>" and closing tag " </tagname>"

2. unpaired Tags: Unpaired tag means it have only single closing tag " </tagname>" or it can be write as <tagname>
