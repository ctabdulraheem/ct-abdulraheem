What is Git?

Git is a version control which is used by developers on daily bases

What is version control?

It means that tracking the code updated. For example on one project 4 developers are working and we need to track which developer is updating what in our project we can control these all things.

Terms to be remember?

• Director -> Folder
• Terminal or Command Line -> Interface for Text Commands
• CLI -> Command Line Interface
• Cd -> Change Directory
• Code Editor -> Word processor for writing Code
• Repository -> Project, or the folder/place where we kept our project
• GitHub is a place we host our repositories online

Git commands?

Git commands? • Clone -> It mean bring hosted repository for online to our local machine
• Add -> Track your file and changes in Git
• Commit -> save your file in Git
• Push -> it is a git commit to upload our repo remotely to GitHub,
